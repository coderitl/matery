###  (第一次使用其他分支)关于分支操作理解:

> 1.  克隆 `git clone -b 分支名 分支地址`
>
> 2.  添加文件，删除文件、修改文件
>
> 3.  `git 管理`
>
>    ```bash
>    
>    git add . # 添加所有文件
>    
>    git commit -m "日志信息" # 添加日志信息
>    
>    git push origin 分支名 # 
>    
>    ```
>
